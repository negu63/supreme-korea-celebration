<br>
<p align="center">
2023년, 드디어 한국에 슈프림 매장이 생길 가능성이 높다는 기쁜 소식에 <br>슈프림의 오랜 팬으로서 제작한 축전 페이지입니다. 
<br>슈프림은 매장을 오픈하면 트레이드 마크인 '박스 로고' 디자인을 사용해 <br>해당 매장 고유의 제품을 출시합니다.
<br>이를 통해 1994년부터 지금까지, 슈프림 매장의 역사를 살펴볼 수 있습니다.
</p>
